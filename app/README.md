
### Описание проекта

Простое одностраничное React-приложение для конвертации валют.

### Функционал

- Выбор исходной и итоговой валюты

- Автоматический пересчет значения

- Кнопка обмена валют местами

- Отображение популярных валютных пар и их текущих значений


### Особенности реализации

- Проект построен на функциональных компонентах React

- Используется локальный стейт и мемоизация результата

- Компоненты разделены по папкам: Converter, Search, Popular

## Запуск
### Клонируйте репозиторий

```bash
git clone https://github.com/Eeegggooorrr1/tip_kr_3
```

### Перейдите в папку проекта

```bash
cd app
```

### Установите зависимости

```bash
npm install
```

### Запустите локальный сервер

```bash
npm run dev
```
